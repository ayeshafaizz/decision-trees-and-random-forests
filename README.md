# DECISION TREES AND RANDOM FOREST

## OVERVIEW

This project demonstrates the implementation of **Decision Tree** and **Random Forest** models using Scikit-learn. It covers data preprocessing, model training, prediction, and performance evaluation for machine learning tasks.

## OBJECTIVES

* Build and train a Decision Tree model.
* Build and train a Random Forest model.
* Compare the performance of both models.
* Evaluate results using appropriate metrics.
* Understand feature importance and model behavior.

## TECHNOLOGIES USED

* Python
* Pandas
* NumPy
* Scikit-learn

## CONCEPTS COVERED

* Decision Trees
* Random Forests
* Data Preprocessing
* Train-Test Split
* Model Evaluation
* Feature Importance

## CONCLUSION

This notebook demonstrates how tree-based machine learning algorithms can be used for prediction tasks. It highlights the strengths of Random Forest over a single Decision Tree and provides practical experience in building, evaluating, and interpreting ensemble models.
